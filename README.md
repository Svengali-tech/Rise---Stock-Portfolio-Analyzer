# Rise - Stock Portfolio Analyzer

A stock portfolio tracking and analysis app built with SwiftUI.

## Features

- 📊 Track your stock holdings with real-time data
- 📈 Visualize portfolio performance with charts
- 💰 Monitor total portfolio value and gains/losses
- ➕ Add and manage multiple stock positions
- 🎨 Modern SwiftUI interface with smooth animations

## Installation

1. Clone the repository
2. Open `StockPortfolioAnalyzer.xcodeproj` in Xcode
3. Build and run on iOS simulator or device

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+
